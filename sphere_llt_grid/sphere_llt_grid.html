sphere_llt_grid

sphere_llt_grid, a Python code which uses longitudes and latitudes to create grids of points, lines, and triangles over the surface of a sphere in 3D.

Languages:

sphere_llt_grid is available in a C version and a C++ version and a Fortran90 version and a MATLAB version and an Octave version and a Python version.
