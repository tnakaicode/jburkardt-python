humps

humps, a Python library which evaluates the humps function, its first and second derivatives, and its antiderivative.

Languages:

humps is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.

Related Data and Programs:

humps_test

Source Code:

  • humps_antideriv.m, returns the antiderivative of the humps function.
  • humps_deriv.m, returns the derivative of the humps function.
  • humps_deriv2.m, returns the second derivative of the humps function.
  • humps_fun.m, returns the humps function.
  • humps_ode.m, returns the derivative of the humps function, suitable for use by ODE solvers.
