circle_distance
Distance Between Random Points on the Unit Circle

circle_distance, a Python library which considers the problem of describing the typical value of the distance between a pair of points randomly selected on the circumference of the unit circle.

Languages:

circle_distance is available in a MATLAB version and a Python version.
