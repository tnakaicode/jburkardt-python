FILUM
File Utilities

FILUM, a Python library which can work with information in files.

Licensing:

The computer code and data files described and made available on this web page are distributed under the GNU LGPL license.

Languages:

FILUM is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.

Related Data and Programs:

FILE_NAME_SEQUENCE, a Python program which demonstrates ways to generate a sequence of filenames that can be used when generating images for an animation.

Source Code:

  • filum.py, the source code;
  • filum.sh, runs all the tests;
  • filum.txt, the output file;

  • filum_prb_4by5.txt, a file with missing right parentheses.
  • i4mat_write_test.txt, a file created by the test for I4MAT_WRITE.
  • r8mat_write_test.txt, a file created by the test for R8MAT_WRITE.
