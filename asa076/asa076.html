asa076

asa076, a Python code which evaluates Owen's T function, by Young and Minder.

This is a version of Applied Statistics Algorithm 76.

Licensing:

Languages:

asa076 is available in a C version and a C++ version and a Fortran90 version and a MATLAB version and an Octave version and a Python version.

Related Data and Programs:

asa005, a Python code which evaluates the CDF of the noncentral T distribution, and uses Owen's T function.

asa243, a Python code which evaluates the CDF of the noncentral T distribution.

owen, a Python code which evaluates Owen's T function.

Author:

Original Fortran77 version by JC Young and Christoph Minder; Python version by John Burkardt.
