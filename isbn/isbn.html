ISBN
Check Digit for International Standard Book Number

ISBN, a Python library which can compute the check digit for an International Standard Book Number (ISBN), or report whether a given ISBN is valid.

Licensing:

The computer code and data files made available on this web page are distributed under the GNU LGPL license.

Languages:

ISBN is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.

Related Data and Programs:

BANK, a Python library which can compute the check digit associated with a US Bank Routing Number check digit, or it can report whether a 9-digit code is actually valid.

LUHN, a Python program which computes the Luhn check digit for a string, and can validate a string.

UPC, a Python library which can determine the check digit for a Uniform Product Code (UPC) or report whether a given UPC is valid.

Source Code:

  • isbn.py, the source code.
  • isbn.sh, runs all the tests.
  • isbn.txt, the output file.
