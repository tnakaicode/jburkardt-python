SQUARE_MONTE_CARLO
Monte Carlo Estimate of Integrals Over Unit Square in 2D

SQUARE_MONTE_CARLO, a Python library which estimates the integral of a function over the interior of the unit square in 2D.

Licensing:

The computer code and data files described and made available on this web page are distributed under the GNU LGPL license.

Languages:

SQUARE_MONTE_CARLO is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.
