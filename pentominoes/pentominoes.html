PENTOMINOES
Some Pentomino Utilities

PENTOMINOES, a Python library which provides some utilities for manipulating pentominoes.

A pentomino is a shape formed by connecting 5 unit squares edgewise. Ignoring reflection and rotation, there are 12 distince pentominoes, nicknamed F, I, L, N, P, T, U, V, W, X, Y and Z.

Related Data and Programs:

Source Code:

  • cell_ij_fill.py fills a unit square with a color.
  • pentomino_display.py displays a particular pentomino in a 5x5 grid.
  • pentomino_matrix.py returns a 0/1 matrix defining a particular pentomino.
  • timestamp.py prints the YMDHMS date as a timestamp.

Examples and Tests:

  • pentominoes_test.py, calls all the tests.
  • pentominoes_test.sh, runs all the tests.
  • pentominoes_test.txt, the output file.
