hexahedron_jaskowiec_rule

hexahedron_jaskowiec_rule, a Python code which returns quadrature rules, with exactness up to total degree 21, over the interior of a hexahedron in 3D, by Jan Jaskowiec, Natarajan Sukumar.

The integration region is: 

       0 <= X <= 1
       0 <= Y <= 1
       0 <= Z <= 1.
