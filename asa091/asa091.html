asa091

asa091, a Python code which computes the percentage points of the Chi-Squared probability distribution, by Best and Roberts. This is a version of Applied Statistics Algorithm 91.

Languages:

asa091 is available in a C version and a C++ version and a Fortran90 version and a MATLAB version and an Octave version and a Python version.

Related Data and Programs:

prob, a Python code which evaluates and inverts many probability distibutions.
