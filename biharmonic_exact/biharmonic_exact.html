biharmonic_exact

biharmonic_exact, a Python code which evaluates exact solutions w(x,y) to the biharmonic equation del^2 w = 0 or wxxxx + 2 wxxyy + wyyyy = 0

Three exact solutions are provided here.

If w(x,y) is regarded as a stream function, then the flow (u,v) = ( dw/dy, -dw/dx ) satisfies the continuity equation dudx + dudy = 0.
