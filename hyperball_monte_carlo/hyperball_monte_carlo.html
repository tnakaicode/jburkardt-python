hyperball_monte_carlo

hyperball_monte_carlo, a Python code which estimates the integral of F(X) over the interior of the unit hyperball in M dimensions.

Licensing:

Languages:

hyperball_monte_carlo is available in a C version and a C++ version and a Fortran90 version and a MATLAB version and an Octave version and a Python version.

Related Data and Programs:

hyperball_integrals, a Python code which defines test functions for integration over the interior of the unit hyperball in m dimensions.

python_monte_carlo, a Python code which uses Monte Carlo sampling to estimate areas and integrals.
