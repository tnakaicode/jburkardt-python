knapsack_values

knapsack_values, a Python code which returns some simple examples of the knapsack problem, in which N items are available, each with given value V and weight W. A thief's knapsack can carry no more than K pounds. The thief seeks a selection S of items to carry in the knapsack of maximum total value.

If the code is called repeatedly, it will return several cases with the values N, V, W, K, and S. This information can be used for testing.
