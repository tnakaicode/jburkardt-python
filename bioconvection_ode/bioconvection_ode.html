bioconvection_ode

bioconvection_ode, a Python code which approximates solutions to a system of ordinary differential equations (ODE) which model a bioconvection problem, and which are related to the Lorenz system.

Languages:

bioconvection_ode is available in a MATLAB version and an Octave version and a Python version.

Related Data and codes:

python_ode, a Python code which sets up various ordinary differential equations (ODE).
