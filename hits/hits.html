hits

hits, a Python code which carries out the HITS algorithm for assigning authority and hub indices to a directed network of web pages.

Languages:

hits is available in a MATLAB version and an Octave version and a Python version.

Related Data and Programs:

pagerank, a Python code which illustrates the eigenvalue (power method) and surfer (Markov chain) approaches to ranking web pages.
