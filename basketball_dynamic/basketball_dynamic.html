basketball_dynamic

basketball_dynamic, a Python code which uses dynamic programming to count the ways a given particular score can be achieved in basketball, respecting the order of events.

To be clear, this code only considers the score of one team. Moreover, in reaching a score, the sequence in which points were made is considered significant. Thus, the two sequences 2 + 2 and 2 + 3, which both achieve a score of 5, and contain the exact same values, are counted as different ways of doing so. They certainly would be experienced differently in live action.
