asa121

asa121, a Python code which evaluates the trigamma function, by BE Schneider. This is a version of Applied Statistics Algorithm 121.

Licensing:

The information on this web page is distributed under the MIT license.

Languages:

asa121 is available in a C version and a C++ version and a Fortran90 version and a MATLAB version and a Octave version and a Python version.

Related Data and Programs:

asa103, a Python code which evaluates the digamma function.

test_values, a Python code which contains sample values for a number of special functions.

Author:

Original Fortran77 version by BE Schneider; This version by John Burkardt.

Reference:

  1. BE Schneider,
    Algorithm AS 121: Trigamma Function,
    Applied Statistics,
    Volume 27, Number 1, 1978, pages 97-99.
