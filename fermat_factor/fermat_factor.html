fermat_factor

fermat_factor, a Python code which uses Fermat's factorization method to decompose an integer N into the product of two factors, N = A^2-B^2 = (A+B)*(A-B).

The algorithm seeks to write N = A^2 - B^2 = ( A + B ) * ( A - B ).
