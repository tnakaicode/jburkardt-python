SAMMON_DATA
Sample M-Dimensional Datasets for Cluster Analysis

SAMMON_DATA, a Python program which generates 6 files of test data for multivariate data clustering.

Usage:

sammon_data

Languages:

SAMMON_DATA is available in a MATLAB version and a Python version.

Source Code:

  • sammon_data.py, the source code.
  • sammon_data.sh, runs the tests.
  • sammon_data.txt, the output file.

  • sammon_circle.txt, points on a circle.
  • sammon_helix.txt, points on a helix.
  • sammon_iris.txt, Fisher's iris data.
  • sammon_linear.txt, points on a line.
  • sammon_nonlinear.txt, points on a nonlinear curve.
  • sammon_simplex.txt, points clustered around the vertices of a simplex.
