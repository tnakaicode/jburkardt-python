NGRAMS
Analyze Occurrence of Ngrams in a String or Test

NGRAMS, a Python library which can analyze a string or text against observed frequences of ngrams (strings of 1, 2, 3, 4, or 5 alphabetic characters.)

Licensing:

Languages:

NGRAMS is available in a Python version.

Related Data and Programs:

GERMAN, a dataset directory which contains some short German texts;

NGRAMS, a dataset directory which contains information about the observed frequency of "ngrams" (particular sequences of n letters) in English text.

TEXT, a dataset directory which contains some short texts in English;

WORDS, a dataset directory which contains lists of words;

Source Code:

  • ngrams.py, the library.
  • ngrams.sh, a BASH shell script that executes the test program.
  • ngrams.txt, the output file.

  • desiderata.txt, a medium length text file.
  • HELLOWORLD.txt, a very short text file.
  • english_monograms.txt a datafile for English monograms.
  • english_bigrams.txt a datafile for English bigrams.
  • english_trigrams.txt a datafile for English trigrams.
  • english_quadgrams.txt a datafile for English quadgrams.
  • english_quintgrams.txt a datafile for English quintgrams.
