bird_egg

bird_egg, a Python code which evaluates some formulas for the shapes of bird eggs.

Licensing:

The information on this web page is distributed under the MIT license.

Languages:

bird_egg is available in a MATLAB version and an Octave version and a Python version.

Related Data and Programs:

joukowsky_transform, a Python code which applies the Joukowsky transform to data, mapping between a circle in the complex plane and various airfoil shapes.

naca, a Python code which takes the parameters of certain NACA airfoils and returns the coordinates of a sequence of points that outline the wing shape.

Reference:

  1. Valeriy Narushin, Michael Romanov, Darren Griffin,
    Egg and math: introducing a universal formula for egg shape,
    Annals of the New York Academy of Sciences,
    Special issue, Annals Reports, pages 169-177, 2021.
