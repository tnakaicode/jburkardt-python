profile_interp

profile_interp, a Python code which interpolates (x,y) data that came from tracing the profile of a face.

Licensing:

The information on this web page is distributed under the MIT license.

Languages:

profile_interp is available in a MATLAB version and an Octave version and a Python version.

Related Data and Programs:

hand, a Python code which carries out some numerical exercises based on data that came from tracing several points on a person's hand.

hand_area, a Python code which estimates the area of a curve which outlines a person's hand.

profile_data, a MATLAB code which returns the (x,y) coordinates of points that came from tracing the profile of a face.

Source Code:

  • profile_interp.py the source code.
  • profile_interp.sh runs all the tests.
  • profile_interp.txt the output file.

  • interp_spline_test.png a plot of the cubic spline interpolant to the profile data.
