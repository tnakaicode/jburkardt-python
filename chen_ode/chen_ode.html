chen_ode

chen_ode, a Python code which sets up and solves the Chen ordinary differential equation (ODE) which has chaotic behavior and an attractor.

Languages:

chen_ode is available in a MATLAB version and an Octave version and a Python version.

Related Data and Programs:

python_ode, a Python code which sets up various systems of ordinary differential equations (ODE).
