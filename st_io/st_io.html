ST_IO
Read and Write Sparse Triplet (ST) Matrix Files.

ST_IO, a Python library which reads and writes files in the Sparse Triplet (ST) sparse matrix format.

Languages:

ST_IO is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.

Related Data and Programs:

ST, the format which is used for the input to this program.

  • st_io.txt, the output file.

  • a5by5_r8.st, an R8ST file storing a 5 by 5 matrix.
  • kershaw_r8.st, an R8ST file storing a 4 by 4 matrix.
