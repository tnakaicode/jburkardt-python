pi_spigot

pi_spigot, a Python code which prints any number of decimal digits of pi.

Licensing:

The information on this web page is distributed under the MIT license.

Languages:

pi_spigot is available in a MATLAB version and an Octave version and a Python version.

Related Programs and Data:

e_spigot, a Python code which can print out the decimal digits of e.

Reference:

  • Stanley Rabinowitz, Stan Wagon,
    A spigot algorithm for the digits of pi,
    American Mathematical Monthly,
    Volume 102, Number 3, pages 195-203, March 1995.

Source Code:

  • pi_spigot.py, the source code.
