PBMB_IO
Portable Bit Map (PBM) binary Files
Read and Write Utilities

PBMB_IO, a Python library which reads and writes files in the binary Portable Bit Map (PBM) format for 2D bit map graphics information.

There are many graphics programs, such as XV and GIMP, that can display images in this format, or translate them to other formats such as BMP, GIF, JPEG, MacPaint, X Window, and so on.

Related Data and Programs:

PBMA_IO, a Python library which handles the ASCII Portable Bit Map (PBM) format.

PBMB, a data directory which contains some example binary Portable Bit Map (PBM) files.

PGMB_IO, a Python library which handles the binary Portable Gray Map (PGM) format.

PPMB_IO, a Python library which handles the binary Portable Pixel Map (PPM) format.
