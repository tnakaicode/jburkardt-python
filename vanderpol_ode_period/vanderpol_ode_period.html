vanderpol_ode_period

vanderpol_ode_period, a Python code which sets up and solves the ordinary differential equations (ODE) defining a van der Pol Oscillator in order to estimate the period of the limit cycle.

Languages:

vanderpol_ode is available in a MATLAB version and an Octave version and a Python version.
