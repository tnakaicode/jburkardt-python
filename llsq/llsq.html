LLSQ
Linear Least Squares (LLS) Solution of Y = A*X+B

LLSQ, a Python library which solves the linear least squares problem of finding the formula of a straight line y=a*x or y=a*x+b which minimizes the root-mean-square error to a set of N data points.

Related Data and Programs:

TEST_LLS, a Python library which implements linear least squares (LLS) test problems of the form A*x=b.
