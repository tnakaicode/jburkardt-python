profile_data

profile_data, a Python code which returns the (x,y) coordinates of points that came from tracing the profile of a face.

Licensing:

The information on this web page is distributed under the MIT license.

Languages:

profile_data is available in a MATLAB version and an Octave version and a Python version.

Related Data and Programs:

house_data, a Python code which returns the coordinates of 11 points which are the vertices of a (degenerate) polygon which has the shape of a house.

profile, a dataset directory which lists the 44 (x,y) coordinates of points traced along the profile of a face.

profile_interp, a Python code which interpolates (x,y) data that came from tracing the profile of a face.

Source Code:

  • profile_data.py the source code.
  • profile_data.sh runs all the tests.
  • profile_data.txt the output file.

  • profile.png the original photograph from which a profile was extracted.
  • profile_data.png a plot of the profile data.
