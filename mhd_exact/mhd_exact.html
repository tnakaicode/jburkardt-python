mhd_exact

mhd_exact, a Python code which evaluates Hartmann's exact formula for a velocity/pressure/magnetic field solution (u,v), p, (b1,b2) of the partial differential equations (PDE) for steady magnetohydrodynamic (MHD) fluid flow in a 2D rectangular channel.
