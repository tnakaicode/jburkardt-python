xy_display

xy_display, a Python code which reads a set of 2D points from a simple XY graphics file and plots them.

Usage:

xy_display ( 'filename.xy' )
where
  • 'filename.xy' is a file containing a list of point coordinates
reads the point coordinate data in 'filename.xy' and displays a plot of the points.
