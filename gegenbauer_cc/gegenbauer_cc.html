gegenbauer_cc

gegenbauer_cc, a Python code which uses a Clenshaw-Curtis approach to approximate the integral of a function f(x) with a Gegenbauer weight.

The Gegenbauer integral of a function f(x) is: 

        value = integral ( -1 <= x <= + 1 ) ( 1 - x^2 )^(lambda-1/2) * f(x) dx
where -0.5 < lambda.
