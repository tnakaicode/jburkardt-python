logistic_bifurcation

logistic_bifurcation, a Python library which plots the bifurcation diagram for the logistic equation, by John Cook.

Languages:

logistic_bifurcation is available in a Python version.

Author:

John Cook

Reference:

  1. John Cook,
    Logistic bifurcation diagram in detail,
    11 January 2020,
    https://www.johndcook.com/blog/

Source Code:

  • logistic_bifurcation.py, the source code.
  • logistic_bifurcation.sh, runs all the tests.
  • logistic_bifurcation.txt, the output file.

  • logistic_bifurcation.png, a plot of the bifurcation diagram.
