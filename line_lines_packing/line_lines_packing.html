line_lines_packing

line_lines_packing, a Python code which tries to randomly pack as many non-overlapping equal line segments as possible into a larger line segment, estimating Renyi's parking constant of 0.7475979202. This is an example of random sequential adsorption (RSA).

When the length of the parking area (larger line segment) is reasonably greater than the length of an individual car (the smaller line segments), then on average about 3/4 of the parking area will be used before there are no more open spots.
