PBMA_IO
ASCII Portable Bit Map Files
Read and Write Utilities

PBMA_IO, a Python library which reads and writes files in the ASCII PBM format for 2D bit map graphics information.

There are many graphics programs, such as XV and GIMP, that can display images in this format, or translate them to other formats such as BMP, GIF, JPEG, MacPaint, X Window, and so on.

Related Data and Programs:

PBMA, a data directory which contains some example ASCII PBM files.

PGMA_IO, a Python library which handles the ASCII PGM (Portable Gray Map) format.

PPMA_IO, a Python library which can read or write ASCII PPM (Portable Pixel Map) graphics files.
