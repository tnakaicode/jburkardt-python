rucklidge_ode

rucklidge_ode, a Python code which approximates solutions to the Rucklidge system of ordinary differential equations (ODE).

Languages:

rucklidge_ode is available in a MATLAB version and an Octave version and a Python version.

Related Data and Programs:

python_ode, a Python code which sets up various systems of ordinary differential equations (ODE).

Reference:

  1. Alastair Rucklidge,
    Chaos in models of double convection,
    Journal of Fluid Mechanics,
    Volume 237, 1992, pages 209-229.
