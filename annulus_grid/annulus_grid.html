annulus_grid

annulus_grid, a Python code which generates a grid of points over the interior of a circular annulus in 2D.

Licensing:

Languages:

annulus_grid is available in a MATLAB version and an Octave version and a Python version.

Related Data and Programs:

annulus_grid_test

python_grid, a Python code which generates a regular grid of points inside a variety of regions in one, two, three or many dimensions.
