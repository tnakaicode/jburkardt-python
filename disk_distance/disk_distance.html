disk_distance

disk_distance, a Python code which considers the problem of describing the typical value of the distance between a pair of points randomly selected in the interior of the unit disk in 2D.

Related Data and Programs:

python_distance, a Python code which estimates the typical distance between a pair of points randomly selected from the surface or interior of a geometric object such as a circle, disk, sphere, cube.
