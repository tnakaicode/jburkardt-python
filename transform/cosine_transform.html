COSINE_TRANSFORM
A Discrete Cosine Transform for Real Data

COSINE_TRANSFORM, a Python library which demonstrates some simple properties of the discrete cosine transform (DCT) for real data.

The code is not optimized in any way, and is intended instead for investigation and education.
