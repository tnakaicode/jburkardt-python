DOOMSDAY
Day of Week for Given Date

DOOMSDAY, a Python library which is given the year, month and day of a date, and uses John Conway's doomsday algorithm to determine the corresponding day of the week.

Licensing:

Languages:

DOOMSDAY is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.

Related Data and Programs:

DATES, a dataset directory which contains lists of dates in various calendar systems.

weekday, a Python library which determines the day of the week for a given day.

weekday_zeller, a Python library which uses Zeller's congruence to determine the day of the week corresponding to a given date, such as 13 July 1989, Gregorian calendar, ... which was a Thursday.
