CUBE_INTEGRALS
Integrals Over the Interior of the Unit Cube in 3D

CUBE_INTEGRALS, a Python library which returns the exact value of the integral of any monomial over the interior of the unit cube in 3D.

The interior of the unit cube in 3D is defined by 

        0 <= X <= 1,
        0 <= Y <= 1,
        0 <= Z <= 1.

The integrands are all of the form 

        f(x,y,z) = x^e1 * y^e2 * z^e3.
where the exponents are nonnegative integers.
