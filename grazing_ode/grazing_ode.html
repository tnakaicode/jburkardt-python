grazing_ode

grazing_ode, a Python code which sets up and solves a pair of ordinary differential equations (ODE) that model the populations of an edible plant, and the herbivore that grazes on it.

In some ways, this problem recalls the predator-prey or Lotka-Volterra ODE, but in this case, there is a stable attracting solution, and strictly positive starting points are gradually attracted to it.
