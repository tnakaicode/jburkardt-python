grf_io

grf_io, a Python code which reads or writes a GRF file containing information about an abstract graph.

The file includes information about

  • adjacency, that is, which nodes are connected to a given node;
  • embedding, that is, how to place each node in the unit square so that a drawing of the graph is visually pleasing and "readable".

Related Data and Programs:

grf, a data directory which contains a description of the GRF format and some examples.
