polynomial_multiply

polynomial_multiply, a Python code which computes the product of two polynomials.

A polynomial p(x) of degree n is represented by a list of n+1 coefficients, so that p(x) = c(0) + c(1) x + c(2) x^2 + ... + c(n) x^(n)
