gyroscope_ode

gyroscope_ode, a Python code which sets up the ordinary differential equations (ODE) that simulate the angular attitude and rotational speed of a gyroscope.

Languages:

gyroscope_ode is available in a MATLAB version and an Octave version and a Python version.

Related Data and codes:

python_ode, a Python code which sets up various systems of ordinary differential equations (ODE).

Reference:

  • Forest Ray Moulton,
    "The Damped Gyroscope",
    Differential Equations,
    Dover, 1958.

Source Code:

  • gyroscope_ode.py, the source code.
  • gyroscope_ode.sh, runs all the tests.
  • gyroscope_ode.txt, the output file.

  • gyroscope_deriv.m, returns the right hand side of the ODE.
  • gyroscope_parameters.m, returns parameters for the ODE.
