vin

vin, a Python code which computes the check digit for a Vehicle Identification Number (VIN), or verifies that a given VIN is legitimate.

Licensing:

The information on this web page is distributed under the MIT license.

Languages:

vin is available in a MATLAB version and an Octave version and a Python version.

Related Data and Programs:

bank, a Python code which computes the check digit associated with a US Bank Routing Number check digit, or reports whether a 9-digit code is valid.

isbn, a Python code which determines the check digit for an International Standard Book Number or reports whether a given ISBN is valid.

luhn, a Python code which computes the Luhn check digit for a string, or validates a string.

upc, a Python code which determines the check digit for a Uniform Product Code (UPC) or reports whether a given UPC is valid.
